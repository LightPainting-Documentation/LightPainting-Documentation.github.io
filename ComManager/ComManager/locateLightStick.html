Locates the LightStick

This method sends a HTTP GET request to the LightStick's locate endpoint. If ipAddress is not specified, it defaults to the lightStick's ipAddress property.

Returns true if the LightStick is located, false otherwise.

Implementation

static Future<bool> locateLightStick({String ipAddress = "default"}) async{
  if (ipAddress == "default"){
    ipAddress = lightStick.ipAddress;
  }
  var url = Uri.parse("http://$ipAddress/locate");
  var client = http.Client();
  var response = await client.get(url);
  if (response.statusCode == 200) {
    return true;
  }
  return false;
}
