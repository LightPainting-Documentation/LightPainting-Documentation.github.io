ComManager

ComManager class Null safety

A class for managing communications with a light stick device.

Provides methods for reading and writing device parameters, system parameters, and image files. Also includes functionality for reading the device's WiFi status and image list. The class is designed to be used as a static class, with methods and properties accessed through the class name rather than through an instance of the class.

lightStick property is an object of CurrentDevice class that stores the data/information regarding the current light stick in use
