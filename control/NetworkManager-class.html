NetworkManager

NetworkManager class Null safety

A class that manages the network-related functionality of the app, such as discovering and tracking active devices on the local network.

It also provides methods for pinging and querying the status of connected devices. The class uses HTTP and UDP protocols for communication with the devices. The activeDevices property is a map of active devices indexed by their IP addresses, while the receiver property is a reference to the active UDP socket used for receiving broadcast messages. The refreshHomePage function can be set by the UI layer to trigger a refresh of the device list when new devices are discovered or removed. The closeApp function is used to close the UDP socket and send a message to all devices to inform them that the app is shutting down.
